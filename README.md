# Amazon E-Commerce Sales Analysis 📊

## Project Overview
This project focuses on analyzing Amazon E-Commerce sales data to uncover valuable business insights and trends. The analysis was performed using Python for data cleaning and preprocessing, while Power BI was used to build an interactive dashboard for visualization.

The dataset used for this project was obtained from Kaggle.

---

## Objectives
- Analyze overall sales performance
- Identify top-performing states and product categories
- Track monthly revenue trends
- Understand order status distribution
- Visualize city-wise sales distribution
- Generate business insights from raw data

---

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook
- Power BI
- Kaggle Dataset

---

## Project Workflow
1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Dashboard Creation
6. Business Insight Generation

---

## Key Insights
- Maharashtra generated the highest sales revenue.
- The "Set" category contributed the most revenue.
- Most orders were successfully shipped.
- Revenue showed fluctuations across different months.
- Major sales were concentrated in metropolitan cities.

- Total Orders & Quantity Tracking
